Si — If Clauses & Conditionals

Spanish si clauses come in two main types: real conditionals (things that are possible or likely) and counterfactual conditionals (things that are hypothetical, contrary to fact, or unlikely). The verb forms in each type follow fixed patterns.

Type 1 — Real conditional (present / future)

Used for conditions that are genuinely possible. The si clause uses the present indicative; the result clause uses the future or present indicative.

Si clauseResult clauseMeaning
si + present indicativefuture tenseIf X happens, Y will happen
si + present indicativepresent indicativeIf X, then Y (general truth)
  • Si tienes tiempo, llámame. If you have time, call me.
  • Si estudias, aprenderás más rápido. If you study, you'll learn faster.
  • Si me siento bien, voy al trabajo. If I feel well, I go to work.

Type 2 — Counterfactual conditional (past hypothetical)

Used for situations that are contrary to current reality — things you wish were true, or imagined scenarios. The si clause uses the imperfect subjunctive; the result clause uses the conditional.

Si clauseResult clauseMeaning
si + imperfect subjunctive (-ra form)conditional (-ría)If X were true, Y would happen

Counterfactual examples with emotion context

  • Si tuviera tiempo, estaría más tranquilo. If I had more time, I would be calmer.
  • Si no tuviera tanto estrés, me sentiría mejor. If I didn't have so much stress, I would feel better.
  • Si pudiera, me quedaría en casa hoy. If I could, I would stay home today.
  • Si fuera más paciente, las cosas irían mejor. If I were more patient, things would go better.
  • Si estuvieras aquí, no estaría tan solo. If you were here, I wouldn't be so lonely.

Key rules to remember

  • Never use si + future or conditional together — this is a common learner error
  • Si quieres ✓ (if you want) — Si querrás ✗
  • The si clause can come first or second in the sentence — meaning stays the same
  • Sí (with accent) = yes — si (no accent) = if
