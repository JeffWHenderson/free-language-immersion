Para — Purpose & Goal

Para and por both translate as "for" in English, but they are not interchangeable. Para points forward — toward a purpose, a destination, or a recipient. Por looks backward — toward a cause, a motive, or an exchange.

The main uses of para

UseStructureExample
Purpose / goalpara + infinitiveEstudio para aprender.
Recipient / intended forpara + noun / pronounEste café es para ti.
Deadlinepara + time expressionLo necesito para el lunes.
Destinationpara + placeSalgo para Madrid mañana.
Opinion / perspectivepara + personPara mí, es difícil.

Para que + subjunctive — "so that"

When the purpose clause has a different subject from the main clause, use para que followed by the subjunctive. If the subject is the same, use para + infinitive instead.

Same subject → para + infinitiveDifferent subject → para que + subjunctive
Hablo despacio para entender.Hablo despacio para que me entiendas.
I speak slowly to understand.I speak slowly so that you understand me.

Para vs. por — quick contrast

ParaPor
Purpose: Estudio para aprender.Cause / motive: Estudio por curiosidad.
Recipient: Un regalo para ti.Exchange: Te lo cambio por este.
Deadline: Terminalo para el viernes.Duration: Estudié por dos horas.
Destination: Salgo para casa.Movement through: Pasé por el parque.

Practical examples

  • Voy al gimnasio para hacer ejercicio. I go to the gym to exercise.
  • Compré flores para mi madre. I bought flowers for my mother.
  • Necesito terminar esto para las cinco. I need to finish this by five o'clock.
  • Te llamo para que no te preocupes. I'm calling you so that you don't worry.
  • ¿Para qué sirve esto? What is this for? / What purpose does this serve?
  • Para mí, lo más importante es la familia. For me, the most important thing is family.
  • Habla más alto para que todos te escuchen. Speak louder so that everyone can hear you.
