Object Pronouns — lo, la, me, te

Object pronouns replace nouns that have already been mentioned, making speech more natural and efficient. Spanish has direct object pronouns (who/what receives the action) and indirect object pronouns (to whom / for whom). Both go before the conjugated verb.

Direct object pronouns

PronounMeaningExample
meme¿Me llamas? (Are you calling me?)
teyou (informal)Te escucho. (I hear you.)
lohim / it (masc.)Lo entiendo. (I understand it.)
laher / it (fem.)La termino hoy. (I finish it today.)
nosusNos ayuda. (He helps us.)
losthem (masc.)Los tengo. (I have them.)
lasthem (fem.)Las envío ahora. (I'm sending them now.)

Indirect object pronouns

PronounMeaningExample
meto/for meMe manda el informe. (She sends me the report.)
teto/for youTe explico todo. (I'll explain everything to you.)
leto/for him/her/you (formal)Le doy el archivo. (I give him the file.)
nosto/for usNos manda un correo. (He sends us an email.)
lesto/for themLes explico el proyecto. (I explain the project to them.)

Placement — before the conjugated verb

  • ¿Lo tienes? Do you have it?
  • Te lo explico ahora. I'll explain it to you now. (two pronouns together)
  • Me lo manda por correo. She sends it to me by email.
  • ¿La terminaste? Did you finish it (fem.)?

With infinitives — attach to the end OR place before

  • Voy a llamarte. / Te voy a llamar. I'm going to call you. (both correct)
  • Necesito explicárselo. / Se lo necesito explicar. I need to explain it to him.

Double pronoun order: indirect before direct

When two pronouns appear together, the indirect pronoun always comes first. When both pronouns start with l (le + lo / les + lo), the indirect changes to se.

  • Te lo mando. (te = indirect, lo = direct) I'll send it to you.
  • Se lo digo. (se replaces le) I'll tell him/her (it). (se = indirect, lo = direct)
