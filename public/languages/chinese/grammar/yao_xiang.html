要 & 想 — Want / Would Like

Both express desire, but with different levels of directness.

要 (yào)想 (xiǎng)
FeelingDirect / firmSoft / tentative
Englishwant / willwould like to / thinking of
Example我要水。(I want water.)我想喝茶。(I'd like to drink tea.)

Structure

Subject + 要/想 + verb/noun

  • 我要米饭。 I want rice. (ordering)
  • 我想吃饺子。 I'd like to eat dumplings.
  • 我想要一杯咖啡。 I'd like a coffee. (想要 = polite want)

Negation

  • 我不要。 I don't want it. (firm refusal)
  • 我不想吃。 I don't feel like eating.
  • 我不要面条，我要米饭。 I don't want noodles, I want rice.

Asking what someone wants

  • 你想吃什么？ What do you want to eat?
  • 你想喝什么？ What would you like to drink?
