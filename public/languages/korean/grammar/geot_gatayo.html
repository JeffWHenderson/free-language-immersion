-(으)ㄹ 것 같아요 — I Think / It Seems

-(으)ㄹ 것 같아요 is used to express a guess, impression, or prediction — "I think (something will happen)" or "it seems like." For past or observed situations, use -(으)ㄴ 것 같아요. Both forms are extremely common in Korean for softening opinions and expressing uncertainty about emotions and events.

Future / Uncertain — -(으)ㄹ 것 같아요

KoreanRomanizationEnglish
슬플 것 같아요seulpeul geot gatayoI think I'll be sad
힘들 것 같아요himdeul geot gatayoIt seems like it'll be tough
괜찮을 것 같아요gwaenchanheul geot gatayoI think it'll be okay
기분이 좋을 것 같아요gibuni joeul geot gatayoI think my mood will be good

Past / Observed — -(으)ㄴ 것 같아요

KoreanRomanizationEnglish
화가 난 것 같아요hwaga nan geot gatayoIt seems like they're angry
슬픈 것 같아요seulpeun geot gatayoIt seems sad / they seem sad
피곤한 것 같아요pigonhan geot gatayoThey seem tired

Example Sentences

  • 오늘 날씨가 추울 것 같아요. oneul nalssiga chuul geot gatayo — I think the weather will be cold today.
  • 그 영화가 재미있을 것 같아요. geu yeonghwaga jaemiisseul geot gatayo — I think that movie will be interesting.
  • 그녀가 외로운 것 같아요. geunyeoga oeRoun geot gatayo — It seems like she's lonely.
  • 별로 행복하지 않을 것 같아요. byeollo haengbokhaji aneul geot gatayo — I don't think I'll be very happy.
