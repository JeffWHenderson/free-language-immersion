숫자 — Two Number Systems

Korean has two entirely separate number systems, and both are used every day. Using the wrong system is a common learner mistake — knowing when to use each one is just as important as knowing the numbers themselves.

Sino-Korean also has: 백 (baek) = 100, 천 (cheon) = 1,000, 만 (man) = 10,000.

When to use each system

Use Pure Korean (고유어) for…Use Sino-Korean (한자어) for…
Age — 살 (sal)Minutes — 분 (bun)
Hours — 시 (si)Dates & months
People — 명 (myeong)Money — 원 (won)
General items — 개 (gae)Floors — 층 (cheung)
AnimalsPhone numbers, years
Before a counter noun, four Pure Korean numbers change form: 하나 → 한  |  둘 → 두  |  셋 → 세  |  넷 → 네

So: 한 명 (one person), 두 개 (two items), 세 살 (three years old) — never 하나 명, 둘 개.

Telling time — mixed system

Hours use Pure Korean; minutes use Sino-Korean. They work together in a single sentence.

  • 두 시 삼십 분 (du si samsip bun) — 2:30 (lit. "two [pure] hour thirty [sino] minute")
  • 열두 시 오 분 (yeoldu si o bun) — 12:05
  • 몇 시예요? (myeot siyeyo?) — What time is it?
  • 한 시예요. (han siyeyo.) — It's 1 o'clock.

Example sentences

  • 나이가 몇 살이에요? (Naiga myeot sarieyo?) — How old are you?
  • 스물다섯 살이에요. (Seumuldaseot sarieyo.) — I'm 25 years old. [Pure Korean]
  • 몇 시예요? (Myeot siyeyo?) — What time is it?
  • 두 시예요. (Du siyeyo.) — It's 2 o'clock. [Pure Korean for hours]
  • 10분 후에 봐요. (Sipbun hue bwayo.) — See you in 10 minutes. [Sino-Korean]
  • 3층이에요. (Samcheungieyo.) — It's the 3rd floor. [Sino-Korean]
